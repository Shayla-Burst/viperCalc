 VERSION
 -------------------------------
 viperCalc version 0.7
 
 
 DESCRIPTION
 -------------------------------
 This is my own personal project I've taken up to learn about GUI programming with Python (specifically, Tkinter), classes, as well as basic math in programming.
 viperCalc is a calculator that prevents error-inducing input being entered, rather than letting it go through and displaying an error on-screen as most calculators do.
 It also supports negative numbers, basic operations, usage of parentheses, and scientific notation. The constant e is not used in this calulator, classic scientific notation (1.0 x 10^9) is instead.
 The calculator uses a combination of strings and eval() to process input; since the user cannot type anything in, 
 and can only use the calculator buttons, eval() is quite safe to use here. However, if this were a command line program, eval() would definitely not be safe to use.
 
 I am taking a break from this project to learn about network programming with Python. However, I will be coming back to it at a later date, when I have more extra time to devote to this.
 On the next version I plan on adding functions for square root, and pi.

I have definitely learned a lot from coding this.
 
 
 REQUIREMENTS
 -------------------------------
 Python 3.0+ must be installed.
 
 
 INSTALL/UNINSTALL/STARTING THE PROGRAM
 -------------------------------
 There is no need to "install" this program. After Python 3.0+ is installed on your device, simply CD over to this folder through the command console, and enter "python viperCalc.py". 
 To uninstall the program, you need only to delete this folder.
